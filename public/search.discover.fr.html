Firefox

A dialog should have appeared that looks like this. Click "Add".

You will need to set this search plugin as your default. From "Preferences" select the "Search" tab, then select kiranico/mh4u from the Default Search Engine list as described below. This will allow you to get search suggestions in your search bar, or to the first result found if typed in your location bar.
